              Web app made for helping people to find solution of their home appliances. If there is any problem in home appliance like fan, refrigerator etc. They can use this web app for finding what the problem with their product is and how to solve it. Web app   provides questions that are related to problem of the product. User can select question and web app will give solution for that specific problem. User will get answer for how to solve their product complaint, cost for repairing and also give suggestion to user about is it possible to repair by user themself or by an expert.and also user and local electricians can buy products from registered stores.
